Overview

Mocha and Moon is a stylish e-commerce website where users can browse trendy products, search for items, and place orders easily. 🛍️☕🌙

Features ✨
🔍 Search Bar
Search products by name.
Dynamic filtering as you type.
Case-insensitive for convenience.
🛒 Product Cards
Each product displayed with image and name.
Automatically hides/shows products based on search input.
📦 Place Order Button & Form
Click Place Order 🛒 to open a form.
Collects Name, Email, Phone, and Address.
Shows a confirmation alert after submission ✅.
📱 Responsive Layout
Works beautifully on both desktop and mobile devices.
Clean, modern design using CSS.
How to Use 📝
Search Products
Type in the search bar 🔍. Matching products appear instantly.
Place Order
Click Place Order 🛒.
Fill in Name, Email, Phone, and Address.
Click Submit Order to see a confirmation message 🎉.
Technologies Used 💻
HTML5 – Structure and layout.
CSS3 – Styling and responsive design.
JavaScript – Search functionality and order form handling.
Notes ⚠️
The search bar works only if each product has a .productcard container with an <h3> title.
The Place Order form currently shows a confirmation alert. Backend integration can be added later to save orders.
